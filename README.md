# Biblioteca
## Sistema(API Rest) para uma biblioteca de escola pública usando Java com Spring, que retorna os livros e usuários da biblioteca.

#### Tecnologias utilizadas
- Java 17
- SpringBoot 3.0.6
- Maven
- Postman 
- h2 
 
 ### Dependências
  - Spring Data Jpa 
  - Spring Web
  - H2 Database
